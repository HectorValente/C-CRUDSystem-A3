User Story 1: Cadastro de TipoCurso

Descrição:

Como usuário, quero poder adicionar, visualizar, modificar e remover tipos de cursos para gerenciar as opções de curso disponíveis.

Critérios de Aceite:

  • Implementação das Funções CRUD:
    • createTipoCurso: Deve gerar um novo objeto TipoCurso com um código único e adicionar ao array.
    • readTipoCurso: Deve permitir a visualização dos detalhes de cada TipoCurso cadastrado.
    • updateTipoCurso: Deve alterar informações de um TipoCurso existente, identificado pelo seu código.
    • deleteTipoCurso: Deve remover um TipoCurso existente do array, identificado pelo seu código.
  • Validação de Código Único:

    Antes da inserção, verificar se o código já existe no array para evitar duplicatas.

  • Operações no Array de TipoCurso:

    Todas as funções CRUD devem interagir com um array central de TipoCurso, realizando as operações necessárias.

  • Feedback ao Usuário:

    Após cada operação, exibir mensagens claras de sucesso ou de erro, informando o resultado da operação.
