User Story 3: Cadastro de Disciplina

Descrição:

Como usuário, quero poder adicionar, visualizar, modificar e remover disciplinas para atualizar a oferta de disciplinas da instituição.

Critérios de Aceite:

  • Implementação das Funções CRUD:
    • createDisciplina: Deve gerar uma nova Disciplina com um código único e vinculá-la a um codigoTipoCurso existente.
    • readDisciplina: Deve permitir a visualização dos detalhes de cada Disciplina cadastrada.
    • updateDisciplina: Deve alterar informações de uma Disciplina existente, identificada pelo seu código.
    • deleteDisciplina: Deve remover uma Disciplina existente do registro, identificada pelo seu código.
  • Vinculação com TipoCurso:

    Cada Disciplina deve estar vinculada a um codigoTipoCurso válido e existente no sistema.

  • Validação de Código Único:

    Antes da inserção, verificar se o código da Disciplina já existe para garantir sua unicidade.

  • Feedback ao Usuário:

    Após cada operação, exibir mensagens de sucesso ou de erro, informando o resultado da ação.
